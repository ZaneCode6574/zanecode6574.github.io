常用控件的使用方法

TextView

  • android: id ="@ + id/ text_ view"
    • 控件ID
  • android: layout_ width =" match_ parent"
    • 宽度：和父布局大小一样，match_parent和fill_parent意义一样，官方推荐使用match_parent
  • android: layout_ height =" wrap_ content"
    • 高度，wrap_content表示刚好包含控件内的内容
  • android: gravity =" center"
    • android:gravity指定对齐方式，可选值有 top、 bottom、 left、 right、 center等，可以用“ |”来同时指定多个值，这里我们指定的 center，效果等同于 center_ vertical | center_ horizontal，表示文字在垂直和水平方向都居中对齐
